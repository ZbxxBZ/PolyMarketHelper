Polymarket 自动交易系统

首次运行配置

第1步：获取私钥

从 MetaMask 或其他钱包导出与 Polymarket 关联账户的私钥。

  1. 打开 MetaMask → 账户详情 → 导出私钥
  2. 输入密码确认
  3. 复制以 0x 开头的私钥字符串

第2步：获取 Funder 地址

这是您在 Polymarket 上的代理钱包（Proxy Wallet）地址。

  1. 访问 polymarket.com 并登录
  2. 进入设置页面或充值页面
  3. 复制显示的存款地址（这是您的 Proxy/Funder 地址）

第3步：配置 .env 文件

在项目根目录创建 .env 文件，填入以下内容：

PRIVATE_KEY=0x你的私钥
FUNDER_ADDRESS=0x你的Funder地址
MONITOR_INTERVAL=30

第4步：重启程序

配置完成后，重新启动 python main.py，即可使用自动交易功能。

安全提示： 私钥仅存储在本地 .env 文件中，不会上传到任何服务器。 请确保 .env 文件已被 .gitignore 忽略。
